# Schema

| Column | Type | Description |
|--------|------|-------------|
| `id` | STRING | PK. A globally-unique ID. |
| `name` | STRING | The order name, generated by combining the order_number with the shop order prefix. |
| `email` | STRING | The email address of the customer. |
| `phone` | STRING | The phone number of the customer. |
| `confirmed` | BOOLEAN | Whether the order is confirmed. |
| `closed` | BOOLEAN | Whether the order is closed. |
| `cancelledAt` | TIMESTAMP | The date and time when the order was cancelled. |
| `cancelReason` | STRING | The reason for cancellation. |
| `currencyCode` | STRING | The currency of the order. |
| `subtotalPrice` | FLOAT | The subtotal price of the order. |
| `totalPrice` | FLOAT | The total price of the order. |
| `totalTax` | FLOAT | The total tax of the order. |
| `totalShippingPrice` | FLOAT | The total shipping price. |
| `totalDiscounts` | FLOAT | The total discounts applied. |
| `discountCodes` | STRING | Discount codes applied to the order as JSON array (code, amount, type). |
| `createdAt` | TIMESTAMP | The date and time when the order was created. |
