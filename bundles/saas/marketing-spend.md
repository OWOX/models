# Schema

| Column | Type | Description |
|--------|------|-------------|
| `spend_id` | STRING | PK. Unique identifier for each spend record. |
| `spend_date` | DATE | Day the spend was incurred. |
| `channel` | STRING | Acquisition channel the spend targeted — attributed to the cohort of accounts acquired through it, not to individual accounts. FK to [Account](./account.md) |
| `campaign` | STRING | Campaign the spend belongs to. |
| `cost` | NUMERIC | Money spent — the CAC numerator. |
| `leads` | INTEGER | Leads generated by the spend. |
| `signups` | INTEGER | Accounts created — CAC denominator. |

# Example Questions

- What is blended and per-channel acquisition cost, and which channels turn leads into signups most efficiently?
- Which channels pay back fastest once their spend is set against the revenue their cohort retains?
- How has cost per signup shifted over time as spend scaled up or down?
